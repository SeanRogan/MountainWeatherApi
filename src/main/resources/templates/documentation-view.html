Peak Conditions API Documentation

endpoint:

/search

GET

search mountain peak by name

Parameter

'query' String - required : the name of the mountain you wish to find

sample responses:

[{"peak_name":"Mount Washington (Nevada)","peak_id":13368},{"peak_name":"Mount Washington (Arizona)","peak_id":13433},{"peak_name":"Mount Washington (New Hampshire)","peak_id":18499},{"peak_name":"Mount Washington (Olympic)","peak_id":19529},{"peak_name":"Mount Washington (British Columbia)","peak_id":19563},{"peak_name":"Mount Washington (Cascades)","peak_id":19650},{"peak_id":19965,"peak_name":"Mount Washington (Oregon)"}]

endpoint:

/report/daily/{peak id}

GET

get one day forecast by mountain id

Parameter

'peak id' String - required : the id String of the peak

Header

'Temp-format' - required : specifies the temperature unit format for weather reports. options are F or C, for Imperial/Fahrenheit units or Metric/Celsius units respectively.
